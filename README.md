# sales-forecasting-ml
This project predicts future monthly sales using Machine Learning. We transform time into a numerical feature and train a Linear Regression model to forecast the next 12 months of sales.

This project demonstrates how to forecast future sales using Machine Learning.

Using the Superstore dataset, we:
• Cleaned and transformed date data  
• Aggregated daily transactions into monthly sales  
• Visualized sales trends and seasonality  
• Converted time into a numerical feature  
• Trained a Linear Regression model to learn the sales trend  
• Forecasted the next 12 months of sales  
• Visualized past vs predicted sales

Tools used:
Python • Pandas • Matplotlib • Scikit-learn
